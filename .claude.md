1. TODO 문서를 진행할 때 단계별로 완성시 문서에 완료 체크를 실행 후 사용자에게 완료한 단계를, 예를 들어 "1단계 완료했습니다. 다음 단계로 넘어갈까요?" 와 같이 고지해야 함.
2. 단계별 구현 후 자체 검증을 진행하고 이상 없을시 '검증완료' 표시함.
3. 단계별 구현 완료 후 필요시 사용자에게 웹화면 확인 단계를 진행해야 함.
4. '히스토리 문서'는 /doc/git-history.md 파일이야.. 이 후 '히스토리 문서 업데이트 요청'하면 이 md 파일 내 규격대로 git commit & push 정보를 업데이트 해야함.
