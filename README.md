# myhousefee

공동주택(다세대, 빌라 등) 관리비 분석 및 추적 시스템

## 기능
- 📸 이미지 업로드 (Claude Vision API 자동 인식)
- 📊 대시보드 (월별 항목별 추이)
- 📈 차트 (비교 및 통계)
- 📋 전체 데이터 테이블 뷰
- 💾 백업/복구 (Excel)
- 📄 PDF 내보내기
- ⚙️ 설정 관리

## 기술 스택
- Frontend: Next.js 16.2.3, React 19.2.4, TypeScript
- 상태 관리: Zustand
- DB: Firebase Firestore
- 배포: Vercel
